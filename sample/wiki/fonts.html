Packaged Fonts

To use fonts, place the file FONT_NAME.ttf (or .otf) into /fonts
then provide the css style "font-family: FONT_NAME"


The following are the default fonts when rendering text.

  • DejaVuSans
  • DejaVuSans-Oblique
  • DejaVuSans-Bold
  • DejaVuSans-BoldOblique
